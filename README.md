# App Gestión Transporte 

Proyecto académico de curso **Plataforma Móviles y Análisis Cloud**.

La aplicación está orientada a la gestión de servicios de transporte para una empresa logística. El flujo principal consiste en el retiro de mercadería desde almacenes temporales y su entrega al cliente final.

## Descripción del proyecto

La app contempla dos tipos de acceso:

- Transportista
- Coordinador de operaciones

Cada tipo de usuario cuenta con pantallas y funcionalidades específicas según su rol dentro del proceso logístico.

## Funcionalidades principales

### Transportista

El transportista puede realizar las siguientes acciones:

- Iniciar sesión en la aplicación.
- Visualizar sus servicios asignados del día.
- Consultar el detalle de un servicio.
- Actualizar el estado de un servicio.
- Registrar evidencia de entrega.
- Revisar el resumen de su jornada.
- Consultar sus notificaciones.
- Visualizar su perfil.

### Coordinador de operaciones

El coordinador puede realizar las siguientes acciones:

- Iniciar sesión en la aplicación.
- Monitorear los servicios de transporte.
- Revisar el estado de los servicios asignados.
- Consultar notificaciones relacionadas a la operación.
- Visualizar su perfil.

## Flujo general

1. El usuario ingresa a la pantalla inicial de selección de acceso.
2. Selecciona si desea ingresar como transportista o como coordinador.
3. Según el tipo de acceso, se muestra el login correspondiente.
4. Luego del inicio de sesión, el usuario accede a las funcionalidades de su rol.
5. El transportista gestiona sus servicios asignados.
6. El coordinador monitorea el avance de los servicios.
